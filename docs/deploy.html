Flutter Deployment Guide

Deploy the transaction classifier ONNX model in a Flutter app using onnxruntime_flutter.

Model Artifact

Fileartifacts/models/teacher.onnx
BackboneEfficientNet-B2
Size29.3 MB (FP32)
Inputimage: float32 tensor [1, 3, 320, 320] (CHW, normalized)
Outputlogit: float32 [1, 1] → apply sigmoid for probability
Threshold0.967 (best F1) or 0.5 (high-recall mode)
Normalizationmean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]

1. Add Dependencies

# pubspec.yaml
dependencies:
  onnxruntime_flutter: ^latest
  image: ^latest

2. Place Model Asset

# pubspec.yaml
flutter:
  assets:
    - assets/models/teacher.onnx

Copy artifacts/models/teacher.onnx into your Flutter project at assets/models/teacher.onnx.

3. Inference Code

import 'dart:math';
import 'dart:typed_data';
import 'package:image/image.dart' as img;
import 'package:onnxruntime_flutter/onnxruntime_flutter.dart';

class TransactionClassifier {
  static const int inputSize = 320;
  static const List<double> mean = [0.485, 0.456, 0.406];
  static const List<double> std = [0.229, 0.224, 0.225];
  static const double threshold = 0.967;

  late OrtSession _session;

  Future<void> init() async {
    final options = OrtSessionOptions();
    _session = await OrtSession.fromAsset('assets/models/teacher.onnx', options);
  }

  Future<Map<String, dynamic>> classify(img.Image image) async {
    final resized = img.copyResize(image, width: inputSize, height: inputSize);
    final input = _preprocess(resized);
    final feeds = {'image': OrtValue.fromList(input, [1, 3, inputSize, inputSize])};
    final results = await _session.run(feeds);
    final logit = (results['logit']!.value as List<List<double>>)[0][0];
    final probability = 1.0 / (1.0 + exp(-logit));
    return {
      'is_transaction': probability >= threshold,
      'confidence': probability,
    };
  }

  Float32List _preprocess(img.Image image) {
    final buffer = Float32List(3 * inputSize * inputSize);
    for (int y = 0; y < inputSize; y++) {
      for (int x = 0; x < inputSize; x++) {
        final pixel = image.getPixel(x, y);
        final idx = y * inputSize + x;
        buffer[0 * inputSize * inputSize + idx] = (pixel.r / 255.0 - mean[0]) / std[0];
        buffer[1 * inputSize * inputSize + idx] = (pixel.g / 255.0 - mean[1]) / std[1];
        buffer[2 * inputSize * inputSize + idx] = (pixel.b / 255.0 - mean[2]) / std[2];
      }
    }
    return buffer;
  }

  void dispose() {
    _session.close();
  }
}

4. Usage Example

final classifier = TransactionClassifier();
await classifier.init();

// From camera or gallery
final imageBytes = await File(pickedPath).readAsBytes();
final image = img.decodeImage(imageBytes)!;
final result = await classifier.classify(image);

if (result['is_transaction']) {
  // Handle transaction image (confidence: ${result['confidence']})
} else {
  // Not a transaction
}

classifier.dispose();

5. Platform Notes

  • Android: ONNX Runtime uses NNAPI delegate automatically on supported devices for hardware acceleration.
  • iOS: ONNX Runtime uses CoreML delegate for GPU/Neural Engine acceleration.
  • Model size: 29.3 MB adds to app bundle. Consider downloading on first launch if app size is critical.
  • Expected latency: 20–40ms on mid-range phones (2023+), 10–20ms on flagships.
  • Memory: ~60–80 MB peak during inference.

6. Threshold Tuning

  • High-recall mode (0.5): catches 94% of transactions but flags some false positives (52% precision).
  • Balanced mode (0.967): best F1=0.74, good balance of precision and recall.
  • Adjust threshold based on UX — if the app auto-categorizes, use high threshold; if it just suggests, use low threshold.
